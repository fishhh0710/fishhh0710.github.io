pA

題意：
給定一張圖，有 個特殊點，要選出第二特殊點，它的定義是：距離其中一特殊點最近的點（所以如果有兩個以上與同一特殊點都是最近，那這兩個就都不是）

作法：

bool spe[120],cho[120]; //特殊節點 第二特殊
vector<pii> edge[120];
void solve(){
    int n,m,k; //點數 邊數 特殊節點數
    cin>>n>>m>>k;
    int a,b,c;
    rep1(i,m){
        cin>>a>>b>>c;
        edge[a].pb({b,c});
        edge[b].pb({a,c});
    }
    rep1(i,k){
        cin>>a;
        spe[a] = 1;
        int mind = 1e9,rec,db = 0; //db->目前最小出現過兩次
        for(pii j:edge[a]){
            if(j.S<mind){
                mind = j.S;
                db = 0;
                rec = j.F;
            }
            else if(j.S==mind){
                db = 1;
            }
        }
        if(!db){
            cho[rec] = 1;
        }
    }
    int ans = 0;
    rep1(i,n){
        ans+=(spe[i]||cho[i]);
    }
    cout<<ans<<"\n";
}

pD

題意：
有 個寶藏，每一個都有價值 。
也有 個守護神，每一個守護神有一個值 代表你拿的寶藏重量不可超過 。
每個寶藏對於每個守護神的重量是不同的，也就是會有一個二維矩陣 代表第 個寶藏對於第 個守護神的重量。

求最多可以拿走多少價值的寶藏。（如果你算出來的可能不是最多，但接近官解的最大值它也會給分）
